# 홈페이지제작 광고 전용 가이드 (2025~2026)

이 문서는 `홈페이지제작` 문의와 구매전환(상담 신청, 견적 문의, 계약)을 늘리기 위한 실행 가이드입니다.  
기존 종합 디자인 광고가 아니라, 홈페이지 제작에만 집중합니다.

---

## 0) 이 문서의 목적과 원칙

- 목표: `문의 수 최대화` + `구매전환 최적화`
- 타겟: B2B + 소상공인 혼합 타겟
- 제작 방식: 바이브코딩/카페24/아임웹 중 고객 맞춤 제안
- 운영 원칙:
  - 노출(많이 보여짐)보다 전환(문의/계약)을 우선
  - 광고 클릭 후 랜딩페이지에서 바로 문의 가능해야 함
  - 채널 성과 평가는 CPA 기준으로 비교

**전문용어 한 줄 정리**
- `CPA (Cost Per Acquisition)`: 전환 1건당 비용 (한 건 문의를 얻는 데 든 광고비)
- `CVR (Conversion Rate)`: 전환율 (클릭한 사람 중 문의한 비율)
- `CTR (Click Through Rate)`: 클릭률 (광고를 본 사람 중 클릭한 비율)
- `ROAS (Return On Ad Spend)`: 광고비 대비 매출 비율

## 1) 홈페이지제작 상품 구조 먼저 확정하기 (광고 전 필수)

광고는 "무엇을 파는지"가 선명할수록 효율이 좋아집니다.

### 1-1. 패키지(Offer) 예시

- `Lite`: 원페이지/소개형 (빠른 오픈)
- `Standard`: 기업형 다페이지 (브랜드/서비스 소개)
- `Commerce`: 쇼핑몰형 (상품, 결제, 운영 기능)
- `Renewal`: 리뉴얼형 (디자인/속도/전환 개선)

### 1-2. 랜딩페이지에 반드시 들어갈 6요소

1. 헤드라인: "무엇을 얼마나 빠르게 해결하는지"
2. 포트폴리오 3~6개 (업종 다양성 포함)
3. 예상 기간/예산 범위 (투명성)
4. 제작 방식 선택지 (바이브코딩/카페24/아임웹)
5. FAQ (유지보수, 수정 횟수, 비용)
6. 문의 폼 + 전화/카카오 CTA 고정

## 2) 제작 방식 선택 가이드 (고객 맞춤 제안용)

한 가지 방식만 강요하지 말고, 고객 목적에 맞춰 추천해야 전환이 올라갑니다.

| 방식 | 적합 고객 | 강점 | 주의점 |
|---|---|---|---|
| 바이브코딩 | 커스텀 요구가 많은 기업/브랜드 | 자유도 높음, 확장성 높음 | 초기 기획 정확도 중요 |
| 카페24 | 쇼핑몰 운영 중심 사업자 | 결제/운영 기능 안정적 | 디자인 커스텀 범위 확인 필요 |
| 아임웹 | 빠른 구축 원하는 소상공인/브랜드 | 구축 속도 빠름, 관리 쉬움 | 고급 기능 확장 시 제약 가능 |

**전문용어 + 쉬운 설명**
- `확장성`: 나중에 기능을 더 붙이기 쉬운 정도
- `커스터마이징`: 고객 요구대로 세부 기능/디자인 수정하는 것
- `운영 안정성`: 결제, 주문, 관리자 기능이 안정적으로 돌아가는 수준

## 3) 채널 우선순위 (전환 중심)

### 실제 광고 효율 우선순위 (초기 예산 기준)
1. `네이버 검색광고(파워링크)`: 국내 홈페이지 제작 수요 대응이 빠르고 문의 전환이 가장 안정적
2. `구글 검색광고`: CPC(클릭당비용)가 상대적으로 낮아 테스트 효율이 좋고 키워드 확장에 유리
3. `메타 리타겟팅`: 한번 방문한 사람 재유입으로 CVR 상승
4. `메타 신규유입(인스타 릴스/캐러셀)`: 포트폴리오 노출로 잠재고객 풀 확대
5. `유튜브`: 신뢰/인지도 보강용 (직접 전환보다 보조 역할)

### 왜 이 순서인가? (비전문가 설명)
- 검색광고는 "이미 찾는 사람"에게 보여서 당장 문의로 이어질 가능성이 높습니다.
- 리타겟팅은 "이미 관심 보인 사람"을 다시 데려오므로 전환비용이 낮아지는 경향이 있습니다.
- SNS 신규유입/유튜브는 즉시 문의보다 "신뢰 형성" 단계 기여가 큽니다.

## 4) 채널별 실행 SOP (비전문가용 순서)

### 4-0. 광고 시작 전 공통 준비 (반드시 선행)

#### A. 랜딩페이지 1개를 먼저 고정합니다
- 주소는 1개만 사용합니다: `포트폴리오 + 문의폼 + 전화/카카오 CTA` 구조
- 메인 홈페이지로 보내지 말고, 광고 전용 랜딩 URL을 따로 둡니다.
- 첫 화면 5초 안에 아래 3가지를 보여줍니다.
  - 무엇을 해주는지(홈페이지 제작/리뉴얼)
  - 어떤 방식이 가능한지(바이브코딩/카페24/아임웹)
  - 어떻게 문의하는지(버튼/폼)

#### B. 전환 추적을 먼저 완성합니다
- 필수 이벤트 3개:
  - `폼 제출 완료`
  - `전화 클릭`
  - `카카오/채팅 클릭`
- 본인 휴대폰으로 직접 3회 테스트합니다.
  - 광고 클릭 -> 랜딩 진입 -> 문의 진행 -> 이벤트 집계 확인

#### C. 예산 분배를 먼저 결정합니다
- 초반 기본값: 검색 70% / 리타겟팅 20% / 신규 SNS 10%
- 이유: 검색이 가장 빠르게 문의를 만들고, 리타겟팅이 전환비용을 낮춰줍니다.

**전문용어 + 쉬운 설명**
- `전환 이벤트`: 문의가 발생했다는 신호를 광고 시스템에 전달하는 설정
- `어트리뷰션`: 어떤 광고가 성과에 기여했는지 배분해보는 방식

## 4-1. 1순위: 네이버 검색광고 (파워링크) 등록 가이드

#### STEP 1) 계정 만들기와 결제 등록
1. `searchad.naver.com` 접속 -> 네이버 계정 로그인
2. 광고 계정 생성 -> 사업자 정보 입력
3. 결제수단 등록(충전식) -> 테스트 예산 먼저 충전

#### STEP 2) 전환추적 설치
1. 광고관리 화면에서 `도구 -> 전환추적` 진입
2. 전환 스크립트 발급 -> 문의 완료 페이지(Thank you)에 삽입
3. 전화/카카오 클릭도 전환으로 잡히게 이벤트 추가
4. 테스트 문의 1회 진행 후 전환 카운트 확인

#### STEP 3) 캠페인/광고그룹 생성
1. 캠페인 생성
   - 이름 예시: `홈페이지제작_검색_전환_250302`
   - 목표: 전환(문의)
2. 광고그룹 3개 생성
   - `홈페이지제작 일반`
   - `홈페이지리뉴얼`
   - `업종별 제작` (병원/학원/쇼핑몰)

#### STEP 4) 키워드 등록 (초보자 기본값)
1. 그룹당 10~20개, 전체 30~50개부터 시작
2. 예시 키워드
   - 홈페이지 제작 업체
   - 홈페이지 제작 비용
   - 홈페이지 리뉴얼
   - 쇼핑몰 홈페이지 제작
   - 병원 홈페이지 제작 업체
   - 학원 홈페이지 제작 견적
3. 제외키워드 등록(예산 누수 방지)
   - 무료, 공짜, 템플릿, 직접 만들기, 취업, 알바, 교육, 강의

#### STEP 5) 광고문구(소재) 등록
1. 제목 작성 공식: `키워드 + 강점 + CTA`
2. 설명 작성 공식: `문제 공감 + 해결 방식 + 상담 유도`
3. 최종 URL은 반드시 광고 전용 랜딩으로 설정

#### STEP 6) 입찰/예산 설정
1. 일예산: 전체 월예산의 40~50% 범위
2. 입찰 시작: 중간 순위(3~5위)부터
3. 첫 3일은 학습 기간으로 큰 변경 금지

#### STEP 7) 7일 후 최적화
1. 클릭은 있는데 문의가 없으면
   - 랜딩 첫 화면 CTA 위치를 위로 올립니다.
   - 문의폼 항목을 3개 이하로 줄입니다.
2. 문의가 나오는 키워드는
   - 입찰가를 소폭 상향
   - 유사 키워드를 3~5개 추가

## 4-2. 2순위: 구글 검색광고 등록 가이드

#### STEP 1) 계정/태그/전환 준비
1. `ads.google.com` 접속 후 계정 생성
2. `도구 및 설정 -> 전환`에서 전환 액션 생성
   - 폼 제출
   - 전화 클릭
   - 카카오 클릭
3. 태그 설치 후 실시간 테스트

#### STEP 2) 캠페인 생성
1. `+ 새 캠페인` 클릭
2. 목표: `리드`
3. 유형: `검색`
4. 네트워크: 검색만 ON, 디스플레이 OFF
5. 지역: 실제 영업 가능 지역으로 한정
6. 입찰전략: `전환수 최대화`

#### STEP 3) 광고그룹과 키워드
1. 네이버와 유사한 그룹 3개로 시작
2. 매치타입 비율
   - `[완전일치]` 50%
   - `"구문일치"` 40%
   - 확장검색 10% (테스트)

#### STEP 4) 반응형 검색광고(RSA) 작성
1. 제목 8~12개 등록
2. 설명 4개 등록
3. 필수 문구 포함
   - 제작 방식 선택 가능(바이브코딩/카페24/아임웹)
   - 업종 맞춤 포트폴리오
   - 무료 상담 CTA

#### STEP 5) 1주 최적화
1. 검색어 보고서에서 불필요 검색어 제외
2. CTR 낮은 제목/설명 교체
3. 전환 발생한 광고그룹 예산 +20% 증액

## 4-3. 3순위: 메타 리타겟팅 등록 가이드

#### STEP 1) 이벤트 수집 점검
1. `이벤트 관리자`에서 픽셀 수집 상태 확인
2. 가능하면 CAPI(전환 API) 같이 연동

#### STEP 2) 맞춤 타겟 만들기
1. 최근 30일 홈페이지 방문자
2. 문의폼 진입 후 미완료 사용자
3. 인스타그램 참여자(프로필 방문/게시물 반응)

#### STEP 3) 캠페인 생성
1. 목표: `리드`
2. 예산: 전체의 15~25%
3. 게재위치: 인스타 피드/스토리/릴스 우선

#### STEP 4) 소재 등록
1. 캐러셀 1개 + 짧은 영상 1개로 시작
2. 메시지 구조
   - 재호출: "아직 고민 중이신가요?"
   - 증거: 사례/포트폴리오
   - 행동유도: "지금 상담하기"

#### STEP 5) 관리 기준
1. 7일 빈도 6회 이상이면 소재 교체
2. CTR은 높고 전환이 낮으면 랜딩 먼저 수정

## 4-4. 4순위: 메타 신규유입(인스타 릴스/캐러셀) 등록 가이드

#### STEP 1) 캠페인 생성
1. 목표: 리드(또는 판매)
2. 타겟: 너무 좁히지 말고 Advantage+ 활용
3. 게재위치: 인스타 릴스/피드/스토리 중심

#### STEP 2) 소재 제작
1. 릴스 15초 2개
   - Before/After
   - 제작 과정 타임랩스
2. 캐러셀 1개
   - 업종별 포트폴리오 3~8장

#### STEP 3) 문구 작성
1. 첫 문장: 고객 문제 제기
2. 둘째 문장: 해결 방식 설명
3. 마지막 문장: 즉시 CTA

#### STEP 4) 7일 후 운영
1. CTR 낮은 소재 OFF
2. 저장/공유/댓글 높은 소재는 예산 증액
3. 전환이 안 나오면 타겟 변경보다 랜딩 개선 먼저

## 4-5. 5순위: 유튜브 광고 등록 가이드 (보조 채널)

#### STEP 1) 시작 조건
1. 검색 + 리타겟팅이 먼저 안정화된 뒤 시작
2. 목표는 직접 전환보다 신뢰 강화

#### STEP 2) 캠페인 구성
1. 6초 범퍼 1개 + 15초 인스트림 1개
2. CTA는 반드시 문의 랜딩으로 연결
3. 리마케팅 대상 중심으로 좁게 운영

#### STEP 3) 평가 기준
1. 유튜브는 단독 CPA보다 보조지표 중심 확인
2. 브랜드 검색량 증가/직접 유입 증가 여부를 함께 체크

## 5) 예산 플랜 (월 10만원부터)

## 5-1. 월 10만원 (초기 검증)
- 목표: 문의 동선 점검 + 키워드 반응 확인
- 배분 예시:
  - 검색광고 70,000원
  - 인스타 30,000원
- 체크포인트:
  - 문의 이벤트 추적 정상 여부
  - 클릭 대비 문의 발생 여부

## 5-2. 월 30만원 (기본 운영)
- 목표: 채널별 승자 찾기
- 배분 예시:
  - 검색광고 180,000원
  - 인스타 90,000원
  - 리타겟팅 30,000원

## 5-3. 월 60만원 (성장 단계)
- 목표: 전환 안정화
- 배분 예시:
  - 검색광고 300,000원
  - 인스타 180,000원
  - 리타겟팅 120,000원

## 5-4. 월 100만원+ (확장 단계)
- 목표: 채널 확장 + CAC 안정화
- 배분 예시:
  - 검색광고 450,000원
  - 인스타 250,000원
  - 리타겟팅 200,000원
  - 유튜브/디맨드젠 100,000원

**전문용어 + 쉬운 설명**
- `CAC (Customer Acquisition Cost)`: 고객 1명 확보 비용

## 7) 주간 운영 루틴 (실무 SOP)

- 매일 5분:
  - 지출 급증/오류 확인
  - 문의 이벤트 정상 수집 확인
- 매주 30분:
  - 검색어 리포트 검토 -> 제외키워드 업데이트
  - 소재별 CTR/CVR 비교 -> 하위 소재 OFF
  - 랜딩 이탈 구간 점검 -> 폼 단순화
- 매월 1시간:
  - 채널별 CPA 비교
  - 고효율 채널에 예산 재배분
  - 새 포트폴리오 반영해 소재 리프레시

## 8) 전환율을 망치는 대표 실수 5가지

1. 광고는 잘 돌리는데 랜딩에 문의 폼이 없음
2. 제작 방식 설명이 없어 고객이 결정을 못함
3. 메인 페이지로 보내서 이탈이 높음
4. 제외키워드 미설정으로 예산 누수 발생
5. 3일 만에 세팅을 자주 바꿔 학습이 깨짐

## 9) 최종 운영 원칙

- 홈페이지제작 광고는 "채널 기술"보다 "오퍼 명확성 + 랜딩 전환 구조"가 먼저입니다.
- 바이브코딩/카페24/아임웹은 경쟁 관계가 아니라, 고객 목적에 맞춰 제안하는 선택지입니다.
- 전환 데이터가 쌓일수록 광고 효율은 좋아집니다. 적은 예산이라도 일관되게 운영하는 것이 핵심입니다.

## 11) 랜딩페이지 문구 템플릿 (섹션별)

아래는 전환형 랜딩에 바로 붙여 쓸 수 있는 문구 템플릿입니다.  
`[ ]` 구간만 서비스에 맞게 교체하면 됩니다.

### 11-1. Hero (첫 화면)
- 헤드라인: "[업종] 홈페이지, [기간] 안에 [핵심성과]까지 설계합니다."
- 서브카피: "바이브코딩·카페24·아임웹 중 목적에 맞는 방식으로 제안드리고, 제작 후 운영까지 고려해 구축합니다."
- CTA 버튼 1: "무료 상담 받기"
- CTA 버튼 2: "포트폴리오 보기"

### 11-2. 신뢰 구간 (Proof)
- 섹션 타이틀: "결과로 검증된 홈페이지 제작"
- 본문: "[누적 프로젝트 수]건 이상의 제작 경험을 바탕으로, 업종별 전환 동선을 표준화해 빠르게 적용합니다."
- 보조문구: "디자인만 예쁜 사이트가 아니라 문의/예약/구매가 발생하는 구조를 만듭니다."

### 11-3. 제작 방식 안내 (Method Fit)
- 섹션 타이틀: "고객 목적에 맞는 제작 방식 제안"
- 카드 1: "바이브코딩 - 맞춤 기능과 확장성을 우선할 때"
- 카드 2: "카페24 - 쇼핑몰 운영 안정성이 중요할 때"
- 카드 3: "아임웹 - 빠른 구축과 쉬운 운영이 필요할 때"
- 보조문구: "상담 시 요구사항을 바탕으로 방식별 장단점과 예상 범위를 함께 안내합니다."

### 11-4. 프로세스 구간 (How It Works)
- 1단계: "요구사항 진단 - 목표/예산/일정 확인"
- 2단계: "전략 제안 - 구조/디자인/제작 방식 확정"
- 3단계: "제작 진행 - 중간 점검과 피드백 반영"
- 4단계: "오픈 및 운영 - 기본 관리 가이드 제공"

### 11-5. 가격/기간 가이드 (Pricing Anchor)
- 타이틀: "예산과 일정, 먼저 투명하게 안내드립니다"
