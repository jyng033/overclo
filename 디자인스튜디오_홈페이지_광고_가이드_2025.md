# 🎨 디자인스튜디오 홈페이지 광고 가이드 (2025~2026 최신)

바이브코딩으로 만든 디자인스튜디오 홈페이지를 광고하기 위한 세 가지 채널별 종합 가이드입니다.

---

## 1) 구글 광고 (Google Ads)

### 2025년 캠페인 유형 (총 7가지)

| 캠페인 유형 | 핵심 용도 | 디자인스튜디오 적합도 |
|---|---|---|
| **검색 캠페인** | 키워드 검색 시 텍스트 광고 노출 | ⭐⭐⭐ 매우 높음 |
| **디스플레이 캠페인 (GDN)** | 6만+ 파트너 사이트에 배너 노출 | ⭐⭐ 높음 |
| **디맨드젠 캠페인** | 유튜브 홈, 디스커버 피드 등 탐색 영역 | ⭐⭐⭐ 매우 높음 |
| **실적 최대화 (P-MAX)** | 모든 구글 인벤토리 올인원 | ⭐⭐ 높음 |
| **동영상 캠페인** | 유튜브 영상 광고 | ⭐⭐ 높음 |
| **쇼핑 캠페인** | 제품 이미지+가격 노출 | ❌ 해당 없음 |
| **앱 캠페인** | 앱 설치 유도 | ❌ 해당 없음 |

### 디자인스튜디오에 추천하는 구글 광고 조합

**① 검색 캠페인 (최우선)**
- 목적: "디자인 외주", "홈페이지 제작 업체", "브랜딩 디자인" 등 검색 시 상위 노출
- 과금: CPC (클릭당 비용) — 클릭할 때만 비용 발생
- 예산: 일일 1~5만원부터 시작 가능
- 핵심 키워드 예시: "웹디자인 스튜디오", "브랜딩 디자인 의뢰", "홈페이지 제작 대행", "로고 디자인 의뢰"
- 팁: 핵심 키워드 + 세부 롱테일 키워드를 조합해 전환당 비용을 낮출 것

**② 디맨드젠 캠페인 (2025년 핵심 변화)**
- 2025년 3월부터 클릭수 최대화 입찰전략 추가 → 전환 태그 없이도 집행 가능
- 유튜브 홈 피드, 구글 디스커버 등 탐색 영역에 이미지/동영상 노출
- 최대 30억 명 도달 가능
- 2025년 하반기 기존 동영상 액션 캠페인(VAC)이 모두 디맨드젠으로 통합 예정
- 디자인 포트폴리오 이미지를 활용한 캐러셀 광고가 특히 효과적

**③ 디스플레이 캠페인 (GDN)**
- 리마케팅 용도로 활용 → 홈페이지 방문 후 이탈한 사용자에게 재노출
- 과금: CPM (1,000회 노출당 비용)
- 배너 사이즈: 반응형 권장 (300×250, 728×90 등)

### 구글 광고 과금 방식 정리

| 과금 방식 | 설명 | 평균 단가 (한국 기준) |
|---|---|---|
| **CPC** (클릭당 비용) | 클릭 시에만 과금 | 업종별 300~3,000원 |
| **CPM** (1,000회 노출당 비용) | 노출 기반 과금 | 1,000~5,000원 |
| **CPA** (전환당 비용) | 구매, 문의 등 전환 시 과금 | 설정에 따라 상이 |
| **CPV** (조회당 비용) | 영상 30초 이상 시청 시 과금 | 50~200원 |

### 2025년 주요 변화 사항
- 실적 최대화 + 검색 캠페인 조합이 "Power Pair"로 가장 효과적
- AI MAX 등 구글 AI 기반 최적화 기능 강화
- 디맨드젠이 GDN 노출까지 확장 → 사실상 디스플레이 + 유튜브 통합 플랫폼으로 진화
- 2025년 3월부터 향상된 CPC(Enhanced CPC) 폐지 → 수동 CPC로 자동 전환

### 🔧 구글 광고 실전 세팅 가이드 (단계별)

**[사전 준비물]**
- Gmail 계정 (구글 계정)
- 광고를 연결할 홈페이지 URL
- 결제 수단 (신용카드/체크카드)
- 광고 소재 (텍스트 카피, 이미지, 로고)

**[STEP 1] 구글 애즈 계정 생성**
1. ads.google.com 접속 → "지금 시작하기" 클릭
2. Gmail 계정으로 로그인
3. "비즈니스 이름" + "홈페이지 URL" 입력
4. 안내에 따라 임시 캠페인을 생성 (나중에 삭제 가능)
5. 결제 정보 입력 → 계정 생성 완료
6. 대시보드에서 임시 캠페인은 "일시중지" 처리

**[STEP 2] 전환 추적 설정 (GA4 + 구글 태그)**
1. 구글 애즈 대시보드 → 상단 "도구 및 설정" → "전환"
2. "새 전환 액션" 클릭 → "웹사이트" 선택
3. 홈페이지 URL 입력 → 자동 스캔
4. 전환 이벤트 설정: "문의하기 클릭", "폼 제출", "전화번호 클릭" 등
5. 구글 태그 코드를 홈페이지 `<head>` 태그 안에 삽입
6. (바이브코딩 홈페이지의 경우) 빌더 설정에서 "헤더 코드 삽입" 영역에 붙여넣기
7. GA4에서 랜딩페이지 이탈 구간 점검 이벤트를 추가:
   - `scroll_50`, `scroll_90` (스크롤 깊이)
   - `form_start`, `form_submit` (문의폼 시작/완료)
   - `click_kakao`, `click_call` (상담 채널 클릭)
8. GA4 탐색(Explore)에서 "랜딩페이지 -> 스크롤 -> 문의폼 시작 -> 제출" 퍼널 리포트를 주 1회 확인
9. ⚠️ 보수적 운영 원칙: "이탈 20% 개선"은 업종/트래픽 품질/랜딩 구조에 따라 달라지므로, 최소 2주 이상 데이터로 판단

**[STEP 3] 검색 캠페인 만들기 (디자인스튜디오 핵심)**
1. 대시보드 → "+ 새 캠페인" 클릭
2. 캠페인 목표: "리드" 또는 "웹사이트 트래픽" 선택
3. 캠페인 유형: "검색" 선택
4. 캠페인 이름: 알기 쉽게 작성 (예: "검색_디자인외주_250302")
5. 네트워크 설정:
   - ✅ 구글 검색 네트워크 (필수)
   - ❌ 검색 파트너 포함 → 해제 권장 (초기에는 불필요)
   - ❌ 디스플레이 네트워크 → 해제 (검색 전용으로 집중)
6. 위치 타겟팅: "대한민국" 또는 서비스 지역 설정
7. 입찰 전략: 초기에는 "전환수 최대화" 선택 → 데이터 축적 후 "타겟 CPA"로 변경
8. 일일 예산: 1~3만원 설정 (최소 1만원)

**[STEP 4] 광고그룹 & 키워드 설정**
1. 광고그룹 이름 설정 (예: "디자인외주_일반")
2. 키워드 입력 — 검색 유형 3가지 활용:
   - `[디자인 외주]` → 일치검색 (정확한 검색어에만 노출)
   - `"홈페이지 제작 업체"` → 구문검색 (구문 포함 시 노출)
   - `브랜딩 디자인` → 확장검색 (관련 검색어까지 확장 노출)
3. 디자인스튜디오 추천 키워드 리스트:
   - [웹디자인 외주], [브랜딩 디자인 의뢰], [홈페이지 제작 대행]
   - "로고 디자인 업체", "UI UX 디자인 스튜디오", "디자인 에이전시"
   - 쇼핑몰 디자인, 앱 디자인 의뢰, 상세페이지 디자인
4. 확장검색 키워드 항목: 초기에는 "사용 안 함" 체크 권장

**[STEP 5] 반응형 검색 광고(RSA) 작성**
1. 광고 제목 (최대 15개 입력, AI가 조합 테스트):
   - "10년 경력 디자인스튜디오"
   - "포트폴리오 무료 상담"
   - "브랜딩부터 웹디자인까지 원스톱"
   - "합리적 비용, 프리미엄 퀄리티"
   - "고객 만족도 95% 이상"
2. 설명 (최대 4개 입력):
   - "전문 디자이너가 맞춤형 디자인을 제공합니다. 지금 무료 상담 신청하세요."
   - "브랜드 아이덴티티부터 웹사이트까지, 디자인스튜디오 [이름]과 함께하세요."
3. 최종 URL: 홈페이지 포트폴리오 페이지 또는 문의 페이지
4. 사이트링크 확장: "포트폴리오 보기", "서비스 소개", "무료 상담", "오시는 길"
5. 콜아웃 확장: "무료 상담 가능", "빠른 납기", "합리적 견적"

**[STEP 6] 캠페인 검토 & 게재 시작**
1. 모든 설정 최종 검토 → "캠페인 게시" 클릭
2. 보통 24시간 이내 광고 심사 완료 → 노출 시작
3. 첫 주: 매일 검색어 보고서 확인 → 불필요한 검색어는 "제외 키워드"로 추가
4. 구글 Ads Editor(무료 앱) 설치 → 대량 수정 시 편리

**[운영 팁]**
- 최소 2주 이상 운영 후 성과 판단 (머신러닝 학습 기간 필요)
- 품질평가점수(QS) 7점 이상 유지 → 키워드·광고문구·랜딩페이지 관련성을 높일 것
- 입찰 전략은 전환 데이터가 30건 이상 쌓이면 "타겟 CPA"로 전환 권장
- 캠페인 이름에 날짜/유형 포함하면 관리가 수월함

## 2) 메타 광고 (인스타그램 + 페이스북)

### 광고 관리 체계

메타 광고는 **캠페인 → 광고세트 → 광고** 3단계 구조입니다.

| 단계 | 설정 내용 |
|---|---|
| **캠페인** | 목표 선택 (인지도, 트래픽, 참여, 리드, 판매 등) |
| **광고세트** | 타겟팅 & 노출 위치 설정 |
| **광고** | 크리에이티브 소재 (이미지, 영상, 텍스트) |

### 캠페인 목표 선택 가이드

| 목표 | 설명 | 디자인스튜디오 활용 |
|---|---|---|
| **인지도** | 브랜드 알리기, 많은 노출 확보 | 스튜디오 런칭 초기 |
| **트래픽** | 홈페이지 방문 유도 | 포트폴리오 페이지로 유입 |
| **참여** | 좋아요, 댓글, 공유 늘리기 | 작업물 인게이지먼트 확보 |
| **리드** | 문의/상담 신청 유도 | 프로젝트 의뢰 확보 |
| **판매** | 직접 구매 전환 유도 | 디자인 패키지 판매 |

### 광고 형식별 특징

| 형식 | 노출 위치 | 디자인스튜디오 활용 팁 |
|---|---|---|
| **이미지 광고** | 피드, 스토리, 탐색 | 대표 포트폴리오 이미지 활용 |
| **동영상 광고** | 피드, 스토리, 릴스 | 작업 과정 타임랩스, Before/After |
| **캐러셀 광고** | 피드, 스토리 | 프로젝트별 슬라이드 구성 |
| **스토리 광고** | 스토리 전용 | 세로형(9:16) 풀스크린 포트폴리오 |
| **릴스 광고** | 릴스 피드 | 15~30초 짧은 작업 영상 |
| **게시물 홍보** | 기존 게시물 확대 | 반응 좋은 포트폴리오 게시물 부스팅 |

### 비용 기준 (2025년 기준)

| 지표 | 페이스북 | 인스타그램 |
|---|---|---|
| **평균 CPC** | 약 1,200원 ($0.87) | 약 1,650원 ($1.15) |
| **평균 CPM** | 약 23,000원 ($16.06) | 약 16,000원 ($10.93) |
| **최소 일일 예산** | 5,000원 | 5,000원 |
| **권장 최소 테스트 기간** | 6일 이상 | 6일 이상 |

### 2025년 핵심 변화: Advantage+ AI 자동 최적화

2025년 초 한국 시장에 AI 기반 광고 최적화 기능 전면 도입:
- **Advantage+ 캠페인**: AI가 자동으로 타겟팅 최적화
- **Advantage+ 크리에이티브**: 소재 자동 개선 (배경 변경, 텍스트 오버레이 등)
- **Advantage+ 타겟팅**: 설정한 타겟 외 전환 가능성 높은 사용자 자동 확장

### 디자인스튜디오 메타 광고 실전 세팅 순서

1. **메타 비즈니스 관리자** 계정 생성 (business.meta.com)
2. **페이스북 페이지** 생성 → 2~3주 활성화 (게시물 꾸준히 올리기)
3. **인스타그램 비즈니스 계정** 연동
4. **메타 픽셀** 설치 → 홈페이지 방문자 추적 & 전환 이벤트 설정
5. **광고 관리자(PC)** 에서 캠페인 생성 (모바일 앱보다 PC 권장)
6. 테스트 기간: 일일 1~3만원 × 6일 이상 진행 후 성과 분석

### 🔧 메타 광고 실전 세팅 가이드 (단계별)

**[사전 준비물]**
- 페이스북 개인 계정 (광고 집행용)
- 스튜디오 브랜드명·로고
- 광고 소재 이미지 (1080×1080 / 1080×1920)
- 결제 수단 (신용카드/체크카드)
- 홈페이지 URL

**[STEP 1] 계정 인프라 구축 (광고 전 1~3주 필수)**

① 메타 비즈니스 관리자 생성:
1. business.meta.com 접속 → "계정 만들기"
2. 비즈니스 이름, 이메일 입력 → 계정 생성
3. 이 관리자에서 광고 계정, 페이지, 픽셀 등을 통합 관리

② 페이스북 페이지 생성:
1. facebook.com → "페이지 만들기"
2. 페이지 이름: 스튜디오 브랜드명 입력 (개인 이름 X)
3. 카테고리: "디자인 서비스" 또는 "웹 디자인 회사"
4. 프로필 사진(로고) + 커버 이미지(포트폴리오) 업로드
5. 게시물 2~3개 이상 등록 (봇 계정 오인 방지)
6. ⚠️ 광고 집행 1~3주 전에 미리 생성 권장 (계정 정지 예방)

- 배경 자동 변경: ON (AI가 다양한 배경 테스트)

**[STEP 6] 게시 & 운영**
1. "게시" 버튼 클릭 → 보통 24시간 이내 심사 완료
2. 심사 거부 시: 사유 확인 후 소재 수정 → 재제출
3. 첫 3~7일: 머신러닝 "학습 기간" → 성과 변동이 큼, 수정 자제
4. 7일 후: 성과 분석 시작

**[운영 & 최적화 팁]**

핵심 지표 모니터링 (광고 관리자 → 열 맞춤 설정):
- CTR (클릭률): 1.5% 이상 목표
- CPC (클릭당 비용): 업종 평균 대비 비교
- 전환율: 리드 기준 2~5% 목표
- 빈도 (Frequency): 3~5회 적정, 7회 이상이면 소재 교체

예산 배분 전략 (퍼널별):
- 초반 (1~2주): 트래픽 캠페인 20% + 전환 캠페인 80%
- 중반 (3~4주): 전환 캠페인 70% + 리타겟팅 10% + 트래픽 20%
- 장기: 전환 70% + 리타겟팅 20% + 인지도 10%

A/B 테스트 방법:
- 같은 광고세트 안에 크리에이티브 2~3개 등록
- 2주 후 CTR·전환율 비교 → 성과 낮은 소재 OFF, 새 소재 추가
- 테스트 요소: 이미지 색감, 카피 문구, CTA 버튼 종류

### 핵심 지표: ROAS (광고비 대비 매출)
- 이커머스 평균 ROAS: 200~300%
- 서비스업(디자인스튜디오) 기준: 프로젝트 1건 수주 금액 대비 광고비로 평가
- 예: 광고비 30만원 → 프로젝트 1건(100만원) 수주 시 ROAS 333%

- 추천: 디맨드젠 캠페인 (2025년 하반기 VAC 통합 후)
- 콘텐츠: 포트폴리오 소개 + CTA 버튼 ("무료 상담 받기")
- 랜딩페이지: 바이브코딩으로 만든 홈페이지의 문의 페이지로 연결

**③ 리마케팅 목적 (재방문 유도)**
- 추천: 비디오 시퀀싱 (1차 브랜딩 → 2차 포트폴리오 → 3차 CTA)
- 홈페이지 방문 후 이탈한 사용자 대상

### 2025년 유튜브 광고 주요 변화

- 쇼츠 광고 단독 캠페인 가능 (노출 형식 직접 선택 가능)

### 🔧 유튜브 광고 실전 세팅 가이드 (단계별)

**[사전 준비물]**
- 구글 애즈 계정 (구글 광고와 동일 계정 사용)
- 유튜브 채널 (브랜드 채널)
- 광고용 동영상 (유튜브에 업로드 완료)
- 결제 수단

**[STEP 1] 유튜브 채널 & 구글 애즈 연결**
1. 유튜브 채널이 없으면 먼저 생성:
   - youtube.com → 로그인 → 우측 상단 프로필 → "채널 만들기"
   - 채널 이름: 스튜디오 브랜드명 사용
   - 프로필 사진(로고), 배너 이미지 설정
2. 광고용 동영상을 유튜브 채널에 업로드:
   - 공개 또는 "미등록" (비공개는 광고 불가)
   - 디자인스튜디오 추천 영상: 15~30초 포트폴리오 하이라이트, 6초 브랜딩 범퍼
3. 구글 애즈 대시보드 → "도구 및 설정" → "연결된 계정"
4. YouTube → "연결" 클릭 → 채널 URL 입력 → 승인
5. 연결 완료 시 유튜브 시청자 데이터를 광고 타겟팅에 활용 가능

**[STEP 2] 동영상 캠페인 만들기**
1. 구글 애즈 대시보드 → "+ 새 캠페인"
2. 캠페인 목표 선택:
   - 브랜드 인지도: "인지도 및 구매 고려도" 선택
   - 홈페이지 유입: "웹사이트 트래픽" 선택
   - 문의 확보: "리드" 선택
   - 세밀한 제어: "목표 따른 안내 없이 캠페인 만들기" (추천)
3. 캠페인 유형: "동영상" 선택
4. 캠페인 하위유형 선택:
   - "동영상 조회수" → 5초 스킵 가능 인스트림 광고 (가장 일반적)
   - "동영상 도달범위" → 범퍼(6초) + 인스트림 조합
   - "효율적 잠재고객 도달" → 범퍼+인스트림+쇼츠 통합 (추천)
5. 캠페인 이름 설정 (예: "유튜브_포트폴리오_인스트림_250302")

**[STEP 3] 예산 & 입찰 설정**
1. 입찰 전략:
   - 조회수 목적 → "최대 CPV" (1회 조회당 최대 입찰가 설정, 예: 100원)
   - 도달 목적 → "목표 CPM" (1,000회 노출당 목표 비용 설정)
2. 일일 예산: 1~3만원 설정
3. 광고 게재 기간: 시작/종료일 설정
4. 네트워크:
   - ✅ YouTube 동영상 (필수)
   - YouTube 검색결과: ON/OFF 선택
   - 디스플레이 네트워크 동영상 파트너: 초기에는 OFF 권장 (유튜브에 집중)

**[STEP 4] 타겟팅 설정**
1. 잠재고객 세그먼트 (맞춤 세그먼트 만들기):
   - "다음을 검색하는 사용자" → 키워드 입력:
     "디자인 외주", "브랜딩 디자인", "홈페이지 제작", "로고 디자인"
   - "다음 URL을 탐색하는 사용자" → 경쟁 스튜디오 웹사이트 URL 입력
   - 이 두 가지를 조합하면 매우 정밀한 타겟팅 가능
2. 인구통계: 연령 25~54세, 성별 모두
3. 위치: 대한민국 (또는 서비스 지역)
4. 콘텐츠 타겟팅 (선택):
   - 주제: "디자인", "비즈니스 서비스", "마케팅"
   - 게재위치: 특정 유튜브 채널 지정도 가능

**[STEP 5] 광고 소재 등록**
1. "동영상 광고 만들기" → 유튜브 영상 URL 붙여넣기
2. 광고 형식 확인:
   - 건너뛸 수 있는 인스트림: 5초 후 스킵, 30초 시청 시 과금 (CPV 80~100원)

## 🔥 전환율 중심 광고 전략: 페이스북 한계와 고효율 대안

### 페이스북 광고, 2025년 현재 한국에서 여전히 유효한가?

#### 📉 페이스북 한국 시장 현황 (냉정한 팩트)

**사용자 이탈이 가속화되고 있다.**

- 18~24세의 70%(32% 크게 줄임 + 38% 다소 줄임)가 페이스북 사용량을 줄였다고 응답
- 25~34세 역시 72%(35% 크게 줄임 + 37% 다소 줄임)가 사용 감소
- 35~44세의 49%는 "더 이상 페이스북이 필요 없다"고 응답
- 한국 페이스북 주 사용층이 40~50대로 이동 완료
- 10~40대 SNS 사용자 수 1위는 **인스타그램** (MAU 2,644만 명, 2025년 2월 기준)

**광고 성과 하락 체감이 뚜렷하다.**

- iOS 14.5 업데이트 이후 전환 추적 정확도가 크게 하락
- 마케터 설문(182건)에서 41.2%가 "전환율 낮아짐"을 1순위로 선택
- CPC/CPM 비용 상승, 도달 수 감소, 전환값 추적 오류 등 복합 문제 발생
- 2030 여성 타겟 캠페인 성과 급감 (iOS 사용자 비중 높은 영향)
- 페이스북 단독 CPM이 3~5만원대로 시작하는 사례가 빈번

#### ✅ 그래도 남아 있는 페이스북의 쓸모

**완전히 버리기보다 "역할을 재정의"해야 한다.**

- Meta 광고 관리자를 통해 광고를 집행하면, 페이스북과 인스타그램에 동시 노출되며 AI가 성과 좋은 지면에 자동 배분한다
- 40~50대 의사결정권자(기업 대표, 임원)가 타겟이라면 페이스북 지면이 여전히 유효
- 리마케팅(방문자 재타겟팅) 캠페인에서는 페이스북 지면의 CPM이 상대적으로 저렴
- Advantage+ 자동 배치를 켜두면 페이스북 지면은 보조 채널로 활용되므로 별도 예산 낭비 없음

**결론: 페이스북 "단독" 광고는 비추천. Meta 광고 내에서 인스타그램 중심 + 페이스북 보조 배치로 운영하는 것이 정답이다.**

### 🎯 전환율 극대화를 위한 채널별 전략 (디자인스튜디오 특화)

> **핵심 원칙**: "많이 노출"보다 "의도가 있는 사람에게 정확히 도달"하는 것이 전환율의 본질

#### ⭐ 1순위: 네이버 검색광고 (파워링크) — 전환율 최고

**왜 전환율이 가장 높은가?**

검색광고는 고객이 **이미 니즈를 가진 상태**에서 직접 검색한 결과에 노출된다. "웹디자인 외주", "로고 디자인 업체"를 검색하는 사람은 이미 디자인 서비스가 필요한 상태이므로 클릭 후 문의(전환)로 이어질 확률이 압도적으로 높다.

**디자인 업종 네이버 검색광고 비용 (2025년 기준)**

| 항목 | 수치 |
|---|---|
| 디자인 업종 CPC (클릭당 비용) | ₩3,000 ~ ₩10,000 |
| 롱테일 키워드 CPC | ₩500 ~ ₩3,000 |
| 최소 일일 예산 | ₩10,000 |
| 예상 전환율 (검색 → 문의) | 3~10% (키워드에 따라 변동) |

**추천 키워드 구성**

```
[핵심 키워드 — CPC 높지만 전환율 높음]
- 웹디자인 외주           // 디자인 서비스 직접 검색 키워드
- 홈페이지 제작 업체       // 홈페이지 제작 의뢰 의도 키워드
- 로고 디자인 의뢰         // 로고 제작 의뢰 의도 키워드
- 브랜딩 디자인 회사       // 브랜딩 서비스 검색 키워드
- 상세페이지 디자인        // 상세페이지 외주 의도 키워드

[롱테일 키워드 — CPC 저렴하고 전환 의도 강함]
- 스타트업 로고 제작       // 소규모 사업자 타겟 키워드
- 쇼핑몰 상세페이지 외주   // 이커머스 사업자 타겟 키워드
- 회사 홈페이지 리뉴얼     // 기존 사이트 개편 의도 키워드
- 패키지 디자인 업체       // 제품 패키지 디자인 의뢰 키워드
- 카페 인테리어 로고       // 특정 업종 맞춤 키워드
- 브랜드 아이덴티티 디자인  // BI/CI 전문 서비스 키워드
```

**네이버 검색광고 세팅 실전 가이드**

**Step 1: 계정 생성 및 기본 설정**
- searchad.naver.com 접속 → 네이버 계정으로 로그인
- 사업자 정보 입력 (개인사업자 또는 법인)
- 결제 수단 등록 (신용카드 또는 무통장입금 — 충전식)
- 광고 관리 시스템(네이버 광고) 대시보드 진입

**Step 2: 캠페인 생성**
- "캠페인 만들기" 클릭
- 캠페인 유형: **파워링크** (검색 결과 상단 텍스트 광고) 선택
- 캠페인 이름: 날짜+목적 포함 (예: "250302_디자인외주_검색")
- 하루 예산 설정: ₩10,000 ~ ₩30,000 (테스트 기간)
- 기간 설정: 시작일/종료일 또는 무기한

**Step 3: 광고 그룹 구성 (핵심!)**
- 키워드를 주제별로 그룹화해야 소재 일치도가 올라가고 품질지수 상승
- 그룹 예시:
  - 그룹1 "홈페이지제작": 홈페이지 제작 업체, 홈페이지 외주, 웹사이트 제작 등
  - 그룹2 "로고디자인": 로고 디자인 의뢰, 로고 제작, CI 디자인 등
  - 그룹3 "브랜딩": 브랜딩 디자인, 브랜드 아이덴티티, BI 디자인 등
- PC 그룹과 모바일 그룹을 분리 운영하면 입찰가 최적화 가능
- 입찰가: 핵심 키워드 ₩3,000~₩5,000 / 롱테일 ₩500~₩2,000 시작

**Step 4: 광고 소재 작성**
- 제목 (15자 이내): 핵심 강점 + 키워드 포함
  - 예: "10년 경력 디자인스튜디오 | 포트폴리오 확인"
  - 예: "홈페이지 제작 전문 | 무료 상담 진행중"
- 설명 (45자 이내): 차별점 + CTA(행동유도)
  - 예: "500+ 프로젝트 완성. 합리적 가격의 맞춤 디자인. 지금 무료 상담 신청하세요"
- 연결 URL: 반드시 포트폴리오 또는 문의 페이지로 연결 (메인 페이지 X)
- 확장소재 추가 (CTR 향상):
  - 서브링크: 포트폴리오 / 서비스 소개 / 상담 신청 / 오시는 길
  - 추가 설명: "무료 상담", "빠른 납기", "합리적 가격" 등

**Step 5: 전환 추적 설정**
- 네이버 광고 > 도구 > 전환추적 > 전환추적 스크립트 발급
- 홈페이지 문의 완료 페이지(Thank you 페이지)에 전환 스크립트 삽입
- 전화 문의 추적: 네이버 광고 전화번호 발급 연동
- 프리미엄 로그 분석 연동으로 유입 경로별 행동 분석 가능

**Step 6: 운영 최적화**
- 첫 1주: 노출/클릭/비용 데이터 수집, 검색어 보고서 확인
- 제외 키워드 등록: "무료", "공짜", "양식", "템플릿" 등 전환 의도 낮은 검색어 차단
- 품질지수 7 이상 유지 목표 (키워드-소재-랜딩페이지 일치도)
- 전환이 발생하는 키워드 발견 시 → 해당 키워드 입찰가 상향 + 유사 키워드 추가
- 요일/시간대별 성과 분석 → 업무 시간(월~금 9~18시) 집중 노출 설정

> **💡 네이버 검색광고 핵심 팁**
> - 핵심 키워드 1~2위 고집하지 말 것 → 3~5위로도 충분히 전환 발생하며 CPC 50% 이상 절감 가능
> - 롱테일 키워드 30개 이상 확보하면 적은 예산으로도 안정적 유입 가능
> - 모바일 비중이 70% 이상이므로 모바일 랜딩 페이지 속도와 문의 동선 반드시 점검
> - 네이버 키워드 도구(keywordtool)에서 핵심 키워드 3~5개 입력하면 연관 키워드 리스트 자동 추출 가능

#### ⭐ 2순위: 인스타그램 릴스 광고 (Meta) — 전환율 + 브랜딩 동시 달성

**왜 디자인스튜디오에 최적인가?**

- 2025년 인스타그램 광고 지출의 50% 이상이 릴스에서 집행됨
- 릴스는 피드 광고 대비 참여율 2~3배 높음
- 79%의 사용자가 릴스 시청 후 제품/서비스를 구매한 경험이 있다고 응답
- 팔로워가 아닌 관심사 기반 신규 사용자에게도 적극 노출 (탐색 알고리즘)
- 디자인 작업물은 시각적 임팩트가 강해 릴스 포맷과 궁합이 뛰어남
- UGC(사용자 생성 콘텐츠) 스타일 릴스는 기존 광고 대비 CPC 50% 낮음 (Impression 데이터)
- 메타 전체 채널 중 구매전환율 50.8%로 1위 기록 (2024년 기준)

**릴스 광고 비용 (2025년 한국 기준)**

| 항목 | 수치 |
|---|---|
| CPC (클릭당 비용) | ₩100 ~ ₩500 |
| CPM (1,000회 노출당) | ₩5,000 ~ ₩15,000 |
| 최소 일일 예산 | ₩5,000 |
| 피드 광고 대비 참여율 | 2~3배 |
| 최소 테스트 기간 | 6일 이상 |

#### ⭐ 3순위: 구글 검색광고 — 네이버와 함께 검색 의도 잡기

기존 가이드의 구글 검색광고 내용을 그대로 활용하되, **전환율 관점에서 추가 최적화 포인트**:

| 강점 | 한국 내 신뢰도 높음 | CPC 저렴, AI 최적화 강력 |

### ⚠️ 전환율 관점에서의 핵심 경고

1. **랜딩 페이지가 전환율의 80%를 결정한다** — 아무리 광고를 잘 세팅해도 랜딩 페이지에 문의 폼이 없거나 로딩이 느리면 전환이 발생하지 않는다. 광고 전에 반드시 모바일 문의 동선 테스트할 것.

2. **페이스북 단독 캠페인은 피할 것** — 2025년 한국 시장에서 페이스북만 타겟하는 광고는 전환 효율이 매우 낮다. Meta 광고 집행 시 인스타그램 지면을 우선 배치하고, 페이스북은 Advantage+ 자동 배치에서 보조 역할로만 활용한다.

3. **"노출" ≠ "전환"** — CPM이 저렴하다고 좋은 게 아니다. 전환당 비용(CPA)을 기준으로 성과를 판단해야 한다. 노출 100만 회보다 전환 10건이 더 가치있다.

4. **검색광고와 SNS 광고를 반드시 병행할 것** — 검색광고는 "이미 필요한 사람"을 잡고, 인스타 릴스 광고는 "아직 필요성을 모르는 잠재 고객"을 발굴한다. 두 채널의 역할이 다르므로 한쪽만으로는 안정적 유입이 어렵다.

5. **리타겟팅은 가장 전환율이 높은 광고다** — 월 예산 중 최소 20%는 리타겟팅에 배분할 것. 홈페이지를 한 번 방문한 사람은 처음 본 사람보다 전환 확률이 3~5배 높다.

6. **AI 챗봇은 "문의 폼 대체"가 아니라 "보완 채널"로 운영할 것** — 실시간 응답으로 야간/주말 이탈 감소 가능성이 있지만, 반드시 사람 상담 전환(전화/카톡/폼 제출) 이벤트까지 연결해 실제 성과를 측정해야 한다.

7. **개인정보/광고 컴플라이언스 점검 없이 추적 확대 금지** — 픽셀, CAPI, GA4, 챗봇을 함께 운영할 때는 개인정보 처리방침 고지와 동의 문구를 최신 상태로 유지해야 광고 계정 리스크를 줄일 수 있다.
